Project Title
Mobile Sales Analytics Dashboard – Power BI

Short Description
This project presents an interactive Mobile Sales Analytics Dashboard built using Microsoft Power BI. It analyzes mobile sales data to provide insights into sales performance, product trends, and regional sales distribution through dynamic visualizations.

Purpose of the Project
The main purpose of this project is to transform raw mobile sales data into meaningful insights that help in understanding sales performance and supporting data-driven business decisions.

Tech Stack
• Microsoft Power BI
• Microsoft Excel (Dataset)
• Data Modeling
• DAX (Data Analysis Expressions)

Key Features
• Interactive dashboard for mobile sales analysis
• Region-wise and product-wise sales insights
• Dynamic filters and slicers for better data exploration
• KPI indicators for total sales, quantity, and performance
• Clean and easy-to-understand visualizations
